# Algorithm Competition Practice Problems

A collection of algorithmic problem solutions implemented in Python for competitive programming practice.

## Overview

This repository contains solutions to various algorithmic challenges commonly found in programming competitions. Each file represents a solution to a specific problem, covering different algorithmic concepts and techniques including:

- **Number Theory**: Modular arithmetic, divisibility problems
- **Dynamic Programming**: Matrix-based DP, recursive solutions
- **Greedy Algorithms**: Optimization problems
- **Binary Search**: Efficient search algorithms
- **String Manipulation**: Stack-based string processing
- **Graph Theory**: Path finding and traversal
- **Combinatorics**: Counting and arrangement problems

## Problem Solutions

### Core Algorithm Problems

- **`yeke_sazi.py`**: Finds the smallest number composed of only 1's that is divisible by a given number
- **`mostatil.py`**: Binary search implementation to find maximum value less than k in a sequence
- **`micromaster.py`**: Sorting algorithm with constrained swap operations using bitwise XOR
- **`matrix.py`**: Dynamic programming solution for matrix path counting
- **`agor_chini.py`**: Matrix-based dynamic programming problem

### Optimization & Greedy Problems

- **`ahmad.py`**: Bin packing problem - fitting objects into boxes with volume constraints
- **`bookshelf.py`**: Bookshelf arrangement optimization problem
- **`baste bandi.py`**: Interval/range optimization problem

### String & Data Structure Problems

- **`reverse string.py`**: Reverses string while preserving punctuation positions using stack
- **`sara.py`**: Recursive backtracking problem

### Mathematical Problems

- **`Algebraic equation.py`**: Quadratic equation solver
- **`dodoyi.py`**: Mathematical sequence problem
- **`kasht_gol.py`**: Fibonacci-like sequence calculation
- **`moorche.py`**: Coordinate-based counting problem

### Object-Oriented Programming

- **`OOP example.py`**: Basic OOP demonstration with inheritance

## Usage

Each Python file can be run independently. Most solutions read input from standard input and output results to standard output, following competitive programming conventions.

### Example

```bash
python yeke_sazi.py
```

## Requirements

- Python 3.x
- NumPy (for `agor_chini.py`)

## Notes

- Some files contain hardcoded test cases for development purposes
- Solutions are optimized for competitive programming constraints
- Code follows competitive programming style (concise, fast execution)

# مسائل تمرینی مسابقات الگوریتم

مجموعه‌ای از راه‌حل‌های مسائل الگوریتمی پیاده‌سازی شده به زبان پایتون برای تمرین برنامه‌نویسی رقابتی.

## نمای کلی

این مخزن شامل راه‌حل‌های چالش‌های الگوریتمی مختلفی است که معمولاً در مسابقات برنامه‌نویسی یافت می‌شوند. هر فایل نشان‌دهنده راه‌حل یک مسئله خاص است که مفاهیم و تکنیک‌های الگوریتمی مختلفی را پوشش می‌دهد از جمله:

- **نظریه اعداد**: حساب پیمانه‌ای، مسائل بخش‌پذیری
- **برنامه‌نویسی پویا**: راه‌حل‌های مبتنی بر ماتریس و بازگشتی
- **الگوریتم‌های حریصانه**: مسائل بهینه‌سازی
- **جستجوی دودویی**: الگوریتم‌های جستجوی کارآمد
- **دستکاری رشته**: پردازش رشته مبتنی بر پشته
- **نظریه گراف**: یافتن مسیر و پیمایش
- **ترکیبیات**: مسائل شمارش و چیدمان

## راه‌حل‌های مسائل

### مسائل الگوریتمی اصلی

- **`yeke_sazi.py`**: پیدا کردن کوچک‌ترین عدد متشکل از فقط رقم 1 که بر عدد داده شده بخش‌پذیر است
- **`mostatil.py`**: پیاده‌سازی جستجوی دودویی برای یافتن حداکثر مقدار کمتر از k در یک دنباله
- **`micromaster.py`**: الگوریتم مرتب‌سازی با عملیات جابجایی محدود با استفاده از XOR بیتی
- **`matrix.py`**: راه‌حل برنامه‌نویسی پویا برای شمارش مسیرهای ماتریسی
- **`agor_chini.py`**: مسئله برنامه‌نویسی پویا مبتنی بر ماتریس

### مسائل بهینه‌سازی و حریصانه

- **`ahmad.py`**: مسئله بسته‌بندی - قرار دادن اشیاء در جعبه‌ها با محدودیت حجم
- **`bookshelf.py`**: مسئله بهینه‌سازی چیدمان کتابخانه
- **`baste bandi.py`**: مسئله بهینه‌سازی بازه/محدوده

### مسائل رشته و ساختار داده

- **`reverse string.py`**: معکوس کردن رشته با حفظ موقعیت علائم نگارشی با استفاده از پشته
- **`sara.py`**: مسئله بازگشتی با backtracking

### مسائل ریاضی

- **`Algebraic equation.py`**: حل‌کننده معادله درجه دوم
- **`dodoyi.py`**: مسئله دنباله ریاضی
- **`kasht_gol.py`**: محاسبه دنباله شبیه فیبوناچی
- **`moorche.py`**: مسئله شمارش مبتنی بر مختصات

### برنامه‌نویسی شی‌گرا

- **`OOP example.py`**: نمایش پایه OOP با ارث‌بری

## نحوه استفاده

هر فایل پایتون می‌تواند به صورت مستقل اجرا شود. اکثر راه‌حل‌ها ورودی را از ورودی استاندارد می‌خوانند و نتایج را به خروجی استاندارد می‌فرستند که مطابق با قراردادهای برنامه‌نویسی رقابتی است.

